# Blog Kerjabaik AI - API Documentation

Dokumentasi API untuk Blog Kerjabaik AI.

## Development

```bash
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000) di browser.

## Deployment

Proyek ini siap di-deploy ke Vercel. Hubungkan repository ini ke Vercel dan deploy otomatis akan berjalan.
